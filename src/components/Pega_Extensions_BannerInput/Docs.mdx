Note on Usage: This component is now available in Pega Infinity 26.1. When configuring page instructions in a view, the referenced Dynamic Text rule includes a new option to display the text as a banner. Once selected, the banner can be displayed as Information, Caution, or Success.

# Overview

The banner input can be used inside a form to display some information to the user. Apply this component to a field of type textarea. The banner is usually display on change on another input field.
